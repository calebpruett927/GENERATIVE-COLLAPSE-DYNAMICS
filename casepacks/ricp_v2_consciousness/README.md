## Purpose

This casepack maps Jackson's 16-stage RICP v2.0 to the GCD consciousness
coherence kernel (8 channels, equal weights 1/8). The v2 extension adds three
new stages (10: Anti-Recursion, 11: Corruption Zone, 12: Signal Womb) between

a trace vector c ∈ [0,1]⁸ and processed through the Tier-1 kernel
K: [0,1]ⁿ × Δⁿ → (F, ω, S, C, κ, IC).

## Key Findings

1. **Re-collapse valley (Stages 10-12).** After reaching Watch at Level 9.0,
   the system drops back into Collapse for three stages before recovering at 13.0.
   This non-monotonic trajectory is the most significant structural difference from v1.

2. **Stage 11 pathology — recursion without return.** The Corruption Zone has the
   worst IC/F ratio (0.896) of any level. The kernel signature: recursive_depth = 0.78
   (second-highest channel) while return_fidelity = 0.15 (lowest). Deep recursion
   with minimal return is corruption by construction. Heterogeneity gap Δ = 0.052
   is 7× larger than any gap in v1. Curvature C = 0.404 is 3× the v1 maximum.

3. **Structural discontinuity from 12.0 → 13.0.** The system jumps from Collapse
   (ω = 0.356) to Watch (ω = 0.078) in one step — the largest single-step change
   in any invariant across both casepacks (ΔF = +0.279). No intermediary levels exist.

4. **No Stable regime reached.** Even Level 13.9 fails the entropy gate
   (S = 0.237 > 0.15) and drift gate (ω = 0.065 > 0.038). The framework
   reaches Watch but never Stable.

5. **Per-channel analysis.** Across 16 levels, cross_scale_coupling has the
   lowest mean (0.469) and information_density the lowest minimum-at-valley
   (0.35 at Stage 10), making them the primary IC killers. All 8 channels
   have minima below 0.20 (at pre-recursive levels).

6. **Cross-version fidelity.** At all 10 shared levels with v1, F and IC match
   to machine precision — the v2 extension preserves the v1 foundation exactly.

7. **All Tier-1 identities verified**: F + ω = 1 (0.0e+00 residual),
   IC ≤ F (all 16 rows), IC = exp(κ) (< 10⁻¹⁵).
