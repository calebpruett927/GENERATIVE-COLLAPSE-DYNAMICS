# Structura Reditus — Object and Field

> **Reditus is the object. Structura Reditus is the field.**

This document fixes the two constitutional layers that sit **above** every acronym in the
corpus. It is the companion to [SUMMA_REDITUS.md](SUMMA_REDITUS.md), which orders the whole
body. Here we define what the corpus is *about* before we describe what it *does*.

---

## The Object — Reditus

**Definition.** *Reditus* is the admissible return of structure through collapse under
declared conditions.

The definition is intentionally strict. A structure has **not** returned merely because it
persists, repeats, resembles itself, restarts, survives in memory, or is repaired by
assertion. It returns only when it re-enters an admissible return domain under the **same
declared rule-set** that governed the collapse boundary.

**Operational form.** Let Φ be a frozen contract and Ψ(t) ∈ [0,1]ⁿ the bounded trace it
admits. A candidate return path

```
Ψ(t₀) → Ψ(t₁) → Ψ(t_R)
```

is *Reditus* only if there exists t_R > t₁ with Ψ(t_R) ∈ D_Φ(t_R) — the admissible return
domain declared by the same contract — with no unauthorized change to the rule-set across
the collapse–return boundary. If no such t_R is observed, the return type is non-return,
τ_R = ∞_rec, and return credit is zero.

**Reditus is not:**

| Phenomenon | Why it is not return |
|------------|----------------------|
| Repetition | The same appearance recurs, but admissible re-entry is not demonstrated. |
| Resemblance | A later structure looks like an earlier one; likeness is not return. |
| Persistence | A structure continues, but continuity did not pass through collapse and re-entry. |
| Repair | A structure is modified after failure, but the repair may alter the rule-set. |
| Recovery | A structure improves after disruption, but improvement is not admissible return. |

Only admissible re-entry under the same declared rule-set satisfies Reditus.

## The Field — Structura Reditus

**Definition.** *Structura Reditus* is the field that studies the structure of admissible
return through collapse.

It studies collapse, continuity, fracture, persistence, transformation, recovery, return,
admissibility, integrity, non-return, seam formation, missingness, and domain translation —
insofar as they bear on the question of return.

**Field discipline.** Because the field is organized around Reditus, it inherits the
discipline of admissible return:

> No appearance without contract. No continuity without return. No return without
> admissibility. No credit without declared structure.

Structura Reditus is not a rhetoric of resilience or a metaphysics of persistence. It is a
**discipline of admissibility**: it asks whether return has been *demonstrated* under
declared conditions, and it preserves both positive results (demonstrated return) and
negative ones (demonstrated non-return, or NON_EVALUABLE when structure is missing).

## Axiom-0 is reflexive — it performs what it names

> *Collapse is generative; only what returns through collapse is real.*

Axiom-0 is not exempt from its own condition. A **dogmatic** axiom is protected by
exemption; a **reflexive** axiom is protected by return. Axiom-0 is reflexive: it both
*describes* Reditus and *performs* Reditus, continuing to return through formalization,
critique, measurement, implementation, translation, closure construction, and domain
expansion without losing its operative burden.

The wording may change across translations; the burden may not. A formulation has not
preserved Axiom-0 if it permits credit without return, treats repetition as return, grants
continuity without seam discipline, or assigns reality without declared admissibility.

> *Axiom-0 is not real within the corpus because it is asserted first. It is real within
> the corpus because it returns.*
